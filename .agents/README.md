This folder contains project-local helper skills for coding agents working on this capstone repository.

Imported commit: `7c5f5607b6cef300d8a2d961363eaccb33c3ea54`

## Folder Map

```text
.agents/
└── skills/
    ├── agent-skills/   # Agent workflow, ADK, security, capstone, eval, and skill-design routers
    ├── excel-skills/   # Excel, spreadsheet, workbook, VBA, and screen-interaction routers
    └── sql-skills/     # Actuarial SQL and ODBC execution routers
```

## How To Use

1. Start with the relevant router `SKILL.md`.
2. Let the router select the smallest focused skill.
3. Read that focused skill's `SKILL.md` before editing, coding, reviewing, or advising.
4. Load deeper references only when the focused skill asks for them.

The root [`../AGENTS.md`](../AGENTS.md) explains when to use these project-local skills.
