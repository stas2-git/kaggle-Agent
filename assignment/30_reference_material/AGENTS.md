# Reference Library Instructions

When using this directory as context:

1. For implementation questions, start with `reference_texts/implementation_specs/README.md`.
2. Read the matching implementation spec before exploring source code.
3. Use `reference_texts/SUMMARY_INDEX.md` to locate supporting course material.
4. Load only the extracted texts routed by that index.
5. Treat `source_documents/` as fallback material for visual checks.
6. Exclude dependency and generated directories from searches.
7. Do not edit original source documents unless the user explicitly requests it.

For capstone implementation, distinguish course requirements from reusable examples: capstone files define what must be delivered; whitepapers explain concepts; codelabs demonstrate possible implementations.
