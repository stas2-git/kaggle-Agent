# Kaggle AI Agents Capstone Project

This repository contains the files and agent implementation for the Actuarial Portfolio Monitoring Agent capstone project.

## Repository Structure

* [capstone_spec_files/](file:///Users/stan/Library/CloudStorage/GoogleDrive-staskhalitov@gmail.com/My%20Drive/keggle%20Agent/capstone_spec_files/): Specifications, backlog, and requirements.
* [refrence material/](file:///Users/stan/Library/CloudStorage/GoogleDrive-staskhalitov%20/My%20Drive/keggle%20Agent/refrence%20material/): Structured course materials and whitepapers.
* [skills/](file:///Users/stan/Library/CloudStorage/GoogleDrive-staskhalitov/My%20Drive/keggle%20Agent/skills/): Custom reusable agent skills (e.g. `extract_pdf_text`).

## Setup

A local python virtual environment has been configured using `uv`.

To sync and activate the environment:
```bash
uv sync
source .venv/bin/activate
```
