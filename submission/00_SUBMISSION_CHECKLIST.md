# Submission Checklist

The local capstone reference says a valid submission needs a Kaggle Writeup, Media Gallery, attached public video, and attached project link.

## Required Kaggle Items

- Kaggle Writeup with title, subtitle, selected track, and project analysis.
- 2,500 words or fewer.
- Media Gallery with a required cover image.
- Public video attached through the Media Gallery.
- Video is 5 minutes or less and published to YouTube.
- Public project link. Use a live demo if available; otherwise use the public code repository with setup instructions.
- Final Kaggle `Submit` button clicked before the deadline.

## Local Files To Review

- Writeup draft: [`01_writeup/kaggle_writeup_draft.md`](01_writeup/kaggle_writeup_draft.md)
- Video script: [`02_video/docs/video_script.md`](02_video/docs/video_script.md)
- Video runbook: [`02_video/docs/demo_runbook.md`](02_video/docs/demo_runbook.md)
- Video assets and generated renders: [`02_video/`](02_video/)

## Suggested Final Pass

1. Confirm the root [`../../README.md`](../../README.md) has clear setup and run instructions.
2. Confirm the writeup explains the problem, agent architecture, implementation, evaluation, and learning journey.
3. Confirm the video shows the product working, not only slides.
4. Confirm links in the Kaggle Writeup are public.
5. Save the Kaggle Writeup, then click `Submit`.
