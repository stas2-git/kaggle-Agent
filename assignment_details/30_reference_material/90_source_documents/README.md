# Original Source Documents

This directory holds binary or rich-text originals.

- `capstone/`: capstone PDFs and RTF/RTFD sources.
- `whitepapers/`: course whitepaper PDFs.

Use the extracted versions in [`../20_searchable_reference_texts/`](../20_searchable_reference_texts/) for normal LLM retrieval. Open originals only to verify extraction, inspect images, or recover formatting-dependent information.
