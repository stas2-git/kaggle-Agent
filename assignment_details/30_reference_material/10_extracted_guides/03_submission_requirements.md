# Submission Requirements

The local capstone reference says the submitted artifact is a Kaggle Writeup with attached media and a public project link.

Read the baseline first: [`00_capstone_baseline_expectations.md`](00_capstone_baseline_expectations.md)

## Required Items

- Kaggle Writeup.
- Media Gallery.
- Attached public video.
- Attached project link.
- Selected track in the Writeup.
- Final `Submit` action before the deadline.

## Writeup

- Treat the Writeup as the project report.
- Keep it at or under 2,500 words.
- Explain the problem, why agents matter, the architecture, how it was built, and how it was evaluated.
- Make the baseline agent qualities visible: tools, safety, evaluation, traces, reproducibility, and deployment awareness.

## Media

- Add a cover image to the Media Gallery.
- Attach the video to the Media Gallery.
- Keep the video to 5 minutes or less.
- Publish the video to YouTube.

## Project Link

- Preferred: public working product or interactive demo.
- Acceptable fallback: public code repository with detailed setup instructions.
